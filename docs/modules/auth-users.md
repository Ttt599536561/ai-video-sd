# 认证与用户

## 用户能力

- 使用邮箱注册。
- 使用邮箱和密码登录。
- 当前不要求邮箱验证码。
- 登录后通过 JWT 调用后端 API。
- 未登录访问用户端视频生成页会跳转到登录注册页。
- 注册成功后自动保存登录状态并进入视频生成页。
- 用户端账号管理页支持修改密码和退出登录。
- 用户端账号管理页当前为白底卡片布局，展示邮箱、注册时间、角色和当前积分；新密码和确认新密码字段对齐。
- 用户端普通用户不显示管理后台入口；入口只有在 `/api/me` 实时确认 `role === "ADMIN"` 后才显示。

## 管理员能力

- 创建首个管理员使用 `BOOTSTRAP_ADMIN_SECRET`。
- 管理员可查看用户邮箱、注册时间、积分、购买套餐和状态。
- 用户管理列表不显示管理员账号，避免后台操作误伤管理员自身。
- 管理员可封禁/解封用户和调整积分；管理端改密仍是后续增强。
- `admin.html` 默认隐藏后台界面，必须通过 `/api/me` 验证管理员角色后才展示；非管理员直接访问会跳回用户端。
- 本地测试管理员：`admin-code-1782584735007@example.com` / `password123`。
- 登录、权限不足和封禁状态已通过后端结构化错误码与前端统一 `apiErrorMessage()` 显示中文提示。
