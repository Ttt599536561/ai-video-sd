# 管理后台

## 当前页面

管理后台在 [admin.html](../../admin.html)，包含模型配置、套餐配置、兑换码、用户管理、视频记录、操作审计和系统设置。

页面默认不展示后台操作内容，必须先调用 `/api/me` 验证当前用户为 `ADMIN` 后才显示；普通用户直接打开 `admin.html` 会被重定向回用户端。

## 能力范围

- 模型配置：模型名称从后端只读供应商模型接口加载为下拉选项，并作为实际供应商模型 ID 保存；模型别名 `displayName` 可由管理员修改，用户前台只展示别名；URL、Key、积分消耗、启用状态仍由管理员配置；操作台默认空，保存即新增；点击编辑后自动填充操作台，保存即更新并清空；支持启用/禁用和删除。
- 模型删除：未被生成记录引用的模型会物理删除；已有历史生成记录引用的模型会软删除并从管理端配置列表和前台模型列表隐藏，历史记录保留。
- 套餐配置：名称、价格、有效期、积分数、排序、状态；操作台默认空，保存即新增；点击编辑后自动填充操作台，保存即更新并清空；支持删除、启用/禁用。
- 兑换码：批量生成、选择数量，输入框默认空，默认 18 位大小写字母+数字，支持永久有效或自定义有效天数；页面采用上下布局，批量生成在上方，全部兑换码记录在下方，记录区固定高度并分页切换；生成成功后弹出本次兑换码弹窗，复制全部只在弹窗内提供，格式为一行一个。管理员复制历史兑换码时复制完整明文码，不复制密文字段。
- 用户管理：邮箱、注册时间、积分、套餐、状态、封禁/解封、改积分；列表不显示管理员账号。
- 视频记录：从后端查看所有用户生成记录，保留 3 天；列表显示用户邮箱、模型展示名、供应商模型 ID、提示词、分辨率、画幅/尺寸、用户选择的视频时长、参考图片/视频/音频数量、扣除积分、状态和“视频生成时长”。用户端另有轻量“生成记录”页，不展示视频地址或供应商 task id。
- 管理端模型和套餐的删除、启用/禁用失败时会显示明确错误，不再表现为点击无反应。
- 操作审计：管理后台新增“操作审计”页签，通过 `/api/admin/audit-logs` 查看最近管理写操作；当前覆盖模型、套餐、兑换码批量生成、用户积分调整和用户封禁/解封。审计 metadata 只保存脱敏摘要，不保存完整模型 Key、兑换码明文或兑换码哈希。
- 系统设置：管理后台新增“系统设置”页签，通过 `/api/admin/system-settings` 读取和保存公网 API 地址。该地址用于供应商抓取参考图片/视频/音频素材，必须是供应商可访问且 HTTPS 证书有效的公网后端地址，例如 `https://api.example.com`。
- 兑换码历史：管理后台通过 `/api/admin/redemption-codes` 读取所有兑换码记录；后端返回完整码、批次、积分、状态、生成时间、兑换信息和 `validityDays`，不返回兑换码哈希。
- 视频记录：管理员可查看生成记录；`generationDurationSeconds` 由完成时间减去任务提交时间得到，单位为秒。用户侧任务记录不可删除，项目页删除视频只影响视频资产记录，不影响生成任务历史。
- 管理后台系统提示统一使用页面正中的弹窗样式；切换模型配置、积分套餐、用户管理、生成记录、操作审计等栏目时不弹系统提示。
